# IE\_7374\_Group13\_Project

North Eastern University IE 7374 Summer group 13 Final project



\# Crash Report Generation Using Llama 3.1



\## Project Overview



This project converts structured Department of Transportation motor carrier

crash records into readable crash summaries and safety recommendations using

the pretrained Llama 3.1 8B Instruct model.



The project evaluates zero-shot, few-shot, and structured prompting strategies

for generating professional crash reports from tabular data.



\## Project Objectives



\- Clean and preprocess commercial motor vehicle crash records

\- Decode categorical crash-condition fields

\- Convert structured records into standardized language-model prompts

\- Generate crash summaries and safety recommendations

\- Evaluate generated reports for accuracy, completeness, and readability
